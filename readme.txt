== Description ==

Let it snow on your WordPress website using Snow. With awesome customizable options, Snow provides you and your visitors the best pixelated snow experience without slowing down your website. You don't need any coding knowledge to use this plugin.

= Fully customizable =
Snow is fully customizable. Change the flake size, color, shadow and a lot more. Dislike the default snowflakes? We got your back. Just upload and use your own snowflake image!

= Schedule and watch it happen =
Snow features a special option which allows you to easily schedule the snowfall!

= Realistic and professional =
You don't want unprofessional scripted snow on your site. No, you want the real deal and you're looking at it right now.

= Lightweight! =
Choose from all kinds of options to speed up your website. Who needs a slow website when you have smart technology?

== Installation ==

1. Upload the 'snow' folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Head over to the Snow menu to get started

== Frequently Asked Questions ==

= How can I reset a specific option to its default value? =

Remove the value from the option field and save the settings.

= How can I reset all options to their default value? =

You want to hold Shift + Alt + R while on a Snow settings tab. A modal will pop up, click 'Reset' to confirm the reset.

= How can I view debugging information? =

You want to hold Shift + Alt + D while on a Snow settings tab. A modal with debugging information will pop up.
